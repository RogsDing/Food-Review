# 美食点评小程序 — 后端管理平台开发计划

## 一、项目背景

当前美食点评微信小程序已具备完整的前端功能（14个页面、22个云函数、6个数据库集合），现在需要搭建一个 **Web 端后台管理系统**，用于管理员对小程序数据进行统一管理。

## 二、技术选型

| 层面 | 选型 | 理由 |
|------|------|------|
| 前端框架 | **Vue 3 + Vite** | 轻量快速，生态成熟 |
| UI 组件库 | **Element Plus** | 国内最流行的中后台 UI 框架，组件丰富，中文友好 |
| 图表库 | **ECharts** | 数据可视化（统计面板） |
| 路由 | **Vue Router 4** | 页面路由与权限控制 |
| 状态管理 | **Pinia** | Vue 3 官方推荐状态管理 |
| 后端接入 | **微信云开发 HTTP API + 新增管理端云函数** | 复用现有云环境数据，无需额外服务器 |
| 部署方式 | **静态站点托管（Vercel / GitHub Pages / 云开发静态托管）** | 纯前端应用，部署简单 |

## 三、功能模块设计

### 模块 1：登录认证
- 管理员账号密码登录
- 登录态保持（Token 存储）
- 登录拦截（未登录自动跳转登录页）
- 退出登录

### 模块 2：仪表盘（Dashboard）
- **核心数据概览卡片**：
  - 总用户数
  - 总点评数
  - 总评论数
  - 总菜谱数
  - 今日新增点评数
  - 今日新增用户数
- **最近动态列表**：最新 10 条点评/评论/留言
- **数据趋势图**：近 7 天/30 天点评量趋势（ECharts 折线图）
- **分类分布图**：各分类点评数量占比（ECharts 饼图）

### 模块 3：点评管理
- **点评列表**（表格展示）：
  - 列字段：店铺名、作者、评分、地区、分类、消费金额、创建时间、操作
  - 支持按店铺名/作者名搜索
  - 支持按分类/地区/评分筛选
  - 支持按时间排序
  - 分页加载
- **查看详情**：弹窗或抽屉展示完整点评信息（含图片预览）
- **删除点评**：管理员可删除任意点评（二次确认）
- **批量操作**：支持批量删除

### 模块 4：评论管理
- **评论列表**（表格展示）：
  - 列字段：内容摘要、所属点评、评论者、创建时间、操作
  - 支持按内容/评论者搜索
  - 支持按关联点评筛选
  - 分页加载
- **删除评论**：管理员可删除任意评论（二次确认）

### 模块 5：留言管理
- **留言列表**（表格展示）：
  - 列字段：内容摘要、留言者、创建时间、操作
  - 支持按内容/留言者搜索
  - 分页加载
- **删除留言**：管理员可删除任意留言（二次确认）

### 模块 6：菜谱管理
- **菜谱列表**（表格/卡片展示）：
  - 字段：名称、分类、难度、评分、浏览量、创建时间、操作
  - 支持按名称/分类搜索
  - 支持按分类/难度筛选
  - 分页加载
- **新增菜谱**：表单填写（名称、分类、菜系、难度、烹饪时间、食材、步骤等）
- **编辑菜谱**：修改已有菜谱信息
- **删除菜谱**：二次确认后删除
- **封面图上传**

### 模块 7：用户管理
- **用户列表**（表格展示）：
  - 列字段：头像、昵称、OpenID、签名、注册时间、点评数、操作
  - 支持按昵称/OpenID 搜索
  - 分页加载
- **查看用户详情**：展示用户基本信息和其发布的所有点评
- **禁用/启用用户**：标记用户状态（可选高级功能）

### 模块 8：反馈管理
- **反馈列表**（表格展示）：
  - 列字段：内容摘要、提交者、提交时间、状态（待处理/已处理）、操作
  - 支持按状态筛选
  - 分页加载
- **查看详情**：完整反馈内容
- **标记已读/已处理**：更新反馈状态

## 五、云函数新增清单

在 `Mini Program/cloudfunctions/` 下新增以下管理端专用云函数：

| 序号 | 云函数名 | 功能说明 | 对应管理页面 |
|------|----------|----------|-------------|
| 1 | `adminLogin` | 管理员密码验证登录，返回 admin token | 登录页 |
| 2 | `getDashboardData` | 聚合统计：用户/点评/评论/菜谱总数、今日新增、近期趋势 | 仪表盘 |
| 3 | `adminGetReviews` | 管理端点评列表查询（支持搜索/筛选/分页，返回全部用户点评） | 点评管理 |
| 4 | `adminDeleteReview` | 管理端删除任意点评（无需权限校验 openId） | 点评管理 |
| 5 | `adminGetComments` | 管理端评论列表查询（含关联点评信息） | 评论管理 |
| 6 | `adminDeleteComment` | 管理端删除任意评论 | 评论管理 |
| 7 | `adminGetMessages` | 管理端留言列表查询 | 留言管理 |
| 8 | `adminDeleteMessage` | 管理端删除任意留言 | 留言管理 |
| 9 | `adminGetUsers` | 管理端用户列表查询（含点评数统计） | 用户管理 |
| 10 | `adminUpdateUser` | 管理端更新用户信息/状态 | 用户管理 |
| 11 | `adminGetRecipes` | 管理端菜谱列表查询 | 菜谱管理 |
| 12 | `adminAddRecipe` | 管理端新增菜谱 | 菜谱管理 |
| 13 | `adminUpdateRecipe` | 管理端编辑菜谱 | 菜谱管理 |
| 14 | `adminDeleteRecipe` | 管理端删除菜谱 | 菜谱管理 |
| 15 | `adminGetFeedback` | 管理端反馈列表查询 | 反馈管理 |
| 16 | `adminUpdateFeedback` | 管理端更新反馈状态（标记已处理） | 反馈管理 |

> **安全说明**：管理端云函数通过在云数据库中新建 `admins` 集合存储管理员账号（账号名 + 密码哈希），`adminLogin` 验证通过后签发 token。后续管理端请求携带 token 进行身份校验。
